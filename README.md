
##  Install

```bash
npm i -D @mergehez/build
npm i -g @mergehez/cli

# yarn 
yarn add -D @mergehez/build
yarn global add @mergehez/cli

# pnpm 
pnpm add -D @mergehez/build
pnpm add -g @mergehez/cli
```
